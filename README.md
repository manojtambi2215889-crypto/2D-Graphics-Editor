# \# 2D Graphics Editor in C

# 

# \## 📌 Project Description

# This project is a \*\*menu-driven 2D Graphics Editor\*\* implemented in C language using a \*\*2D character array\*\* as the drawing canvas.

# 

# The canvas is initially filled with the character `\_` (underscore), and graphical objects are drawn using `\*` (asterisk).

# 

# \---

# 

# \## ✨ Features

# \- Draw Rectangle

# \- Draw Line

# \- Draw Triangle

# \- Display Canvas

# \- Clear Canvas

# \- Modify objects by redrawing

# \- Menu-driven interface for user interaction

# 

# \---

# 

# \## 🧠 Concepts Used

# \- 2D Arrays in C

# \- Functions

# \- Loops and Conditional Statements

# \- Modular Programming

# \- Basic Graphics logic using characters

#
